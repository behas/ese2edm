3. Query

PREFIX owl:  <http://www.w3.org/2002/07/owl#>
PREFIX ore:  <http://www.openarchives.org/ore/terms/>
PREFIX dc:  <http://purl.org/dc/elements/1.1/>
CONSTRUCT { ?RWO_Europena owl:sameAs ?I }
WHERE { 
   ?Proxy_Europeana dc:identifier ?I .
   ?Proxy_Europeana ore:proxyFor ?RWO_Europena .
}

4. replace literal values by URI values for owl:sameAs

sed 's/\<owl\:sameAs\>\(.*\)\<\/owl\:sameAs\>/\<owl\:sameAs rdf\:resource=\"\1\"\/\>/g' < query-result.rdf > SOCH-links.rdf
